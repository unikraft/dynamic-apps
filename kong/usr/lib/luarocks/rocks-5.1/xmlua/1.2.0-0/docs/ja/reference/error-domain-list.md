# `xmlua.HTMLSAXParser.error.domain`と`xmlua.XMLSAXParser.error.domain`の値一覧

`xmlua.HTMLSAXParser.error.domain`と`xmlua.XMLSAXParser.error.domain`の値は以下の通りです。

```
XML_FROM_NONE = 0
XML_FROM_PARSER = 1 : XMLパーサー
XML_FROM_TREE = 2 : treeモジュール
XML_FROM_NAMESPACE = 3 : XML名前空間モジュール
XML_FROM_DTD = 4 : パーサーコンテキストによるXML DTD検証
XML_FROM_HTML = 5 : HTMLパーサー
XML_FROM_MEMORY = 6 : メモリーアロケーター
XML_FROM_OUTPUT = 7 : シリアライゼーションコード
XML_FROM_IO = 8 : Input/Outputスタック 
XML_FROM_FTP = 9 : FTPモジュール
XML_FROM_HTTP = 10 : HTTPモジュール
XML_FROM_XINCLUDE = 11 : XInclude処理
XML_FROM_XPATH = 12 : XPathモジュール
XML_FROM_XPOINTER = 13 : XPointerモジュール
XML_FROM_REGEXP = 14 : 正規表現モジュール
XML_FROM_DATATYPE = 15 : W3C XML Schemas データタイプモジュール
XML_FROM_SCHEMASP = 16 : W3C XML Schemas パーサーモジュール
XML_FROM_SCHEMASV = 17 : W3C XML Schemas 検証モジュール
XML_FROM_RELAXNGP = 18 : Relax-NG パーサーモジュー
XML_FROM_RELAXNGV = 19 : Relax-NG 検証モジュール
XML_FROM_CATALOG = 20 : カタログモジュール
XML_FROM_C14N = 21 : 正規化モジュール
XML_FROM_XSLT = 22 : XSLTエンジン(libxslt)
XML_FROM_VALID = 23 : XML DTD 検証
XML_FROM_CHECK = 24 : エラーチェックモジュール
XML_FROM_WRITER = 25 : xmlwriter モジュール
XML_FROM_MODULE = 26 : 動的ロードモジュール
XML_FROM_I18N = 27 : 文字変換ハンドリングモジュール
XML_FROM_SCHEMATRONV = 28 : Schematron検証モジュール
XML_FROM_BUFFER = 29 : バッファーモジュール
XML_FROM_URI = 30 : URIモジュール
```
